# biostats-sas-examples
Example statistical analysis using SAS for graduate level biostatistics and epidemiology coursework 

This repository contains example statistical analysis completed as part of graduate-level biostatistics and epidemiology coursework. The scripts demonstrate data cleaning, descriptive statistics, hypothesis testing, and regression using SAS.
All datasets are either simulated or publically avaliable (e.g. NHANES). This repository is intended to demonstrate reproducible, script-based analysis workflows.
